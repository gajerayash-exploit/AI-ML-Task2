# AI & ML Internship - Task 2

## Task
Exploratory Data Analysis (EDA)

## Objective
To analyze and understand the dataset using statistics and visualizations.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Titanic Dataset

## Work Performed
- Imported and explored the dataset
- Generated summary statistics
- Checked missing values and data types
- Created histograms for feature distributions
- Visualized outliers using boxplots
- Generated correlation matrix heatmap
- Created pairplots for feature relationships
- Analyzed survival patterns using countplots
- Derived feature-level insights and conclusions

## Key Insights
- Most passengers did not survive.
- Female passengers had higher survival rates.
- Passenger class influenced survival probability.
- Fare and Age contained outliers.
- Correlation analysis revealed useful feature relationships.
